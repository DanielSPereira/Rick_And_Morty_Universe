
# Rick_And_Morty_Universe

Olá, esse repositório contém o código de uma plataforma interativa cujo objetivo é listar os personagens do desenho "Rick and Morty".
Aonde é possível ver informações detalhadas de cada personagem e adicioná-los a uma lista de favoritos.

![image](https://user-images.githubusercontent.com/67342331/171298057-d9069dde-edc3-4cf6-be9b-686661e147d0.png)

## Rodando localmente

Clone o projeto

``
  git clone https://github.com/DanielSPereira/Rick_And_Morty_Universe.git
``
Instale as dependências

> yarn
  
Inicie o servidor
``
  yarn dev
``

## Testando aplicação

Após ter configurado o projeto e instalado as dependências na sua máquina, rode o comando:
``
yarn test
``
para vizualizar os testes, ou

``
yarn watch
``
para vizualizar testes em tempo de desenvolvimento.

## Tecnologias utilizadas

- Vite
- React JS
- Tailwind
- Jest
- Testing Library
- GraphQL
- Apollo

## Funcionalidades 

- Vizualização dos personagens
- Pesquisa por nome dos personagens
- Criar lista de personagens favoritos
- Vizualizar personagens por filtro
